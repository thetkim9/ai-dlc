## Backend Components (Spring Boot)

### 1. AuthController
**목적**: 관리자 인증 및 테이블 세션 인증 처리
**책임**:
- 관리자 로그인/로그아웃 처리
- JWT 토큰 발급 및 검증
- 테이블 자동 로그인 처리 (매장 식별자 + 테이블 번호 + 비밀번호)
- 테이블 세션 토큰 발급

### 2. MenuController
**목적**: 메뉴 및 카테고리 조회/관리 API 제공
**책임**:
- 카테고리별 메뉴 목록 조회 (고객/관리자 공용)
- 메뉴 등록, 수정, 삭제 (관리자 전용)
- 메뉴 이미지 파일 업로드 처리
- 메뉴 노출 순서 조정

### 3. OrderController
**목적**: 주문 생성 및 조회 API 제공
**책임**:
- 고객 주문 생성
- 현재 세션 주문 내역 조회 (고객용)
- 테이블별 주문 목록 조회 (관리자용)
- 주문 상태 변경 (관리자용)
- 주문 삭제 (관리자 직권)

### 4. TableController
**목적**: 테이블 및 세션 관리 API 제공
**책임**:
- 테이블 초기 설정 (번호, 비밀번호)
- 테이블 세션 생성/종료
- 테이블 이용 완료 처리 (주문 이력 이동 + 리셋)
- 과거 주문 내역 조회

### 5. SseController
**목적**: Server-Sent Events 스트림 제공
**책임**:
- 관리자용 SSE 연결 관리 (신규 주문, 상태 변경 이벤트)
- 고객용 SSE 연결 관리 (주문 상태 변경 이벤트)
- SSE 클라이언트 연결/해제 관리

### 6. AuthService
**목적**: 인증 비즈니스 로직 처리
**책임**:
- 관리자 자격증명 검증 (bcrypt)
- JWT 토큰 생성/파싱/검증
- 로그인 시도 횟수 제한 (brute force 방지)
- 테이블 세션 토큰 생성/검증

### 7. MenuService
**목적**: 메뉴 비즈니스 로직 처리
**책임**:
- 메뉴 CRUD 비즈니스 로직
- 이미지 파일 저장/삭제 처리
- 메뉴 순서 조정 로직
- 입력값 유효성 검증

### 8. OrderService
**목적**: 주문 비즈니스 로직 처리
**책임**:
- 주문 생성 및 검증
- 주문 상태 전이 관리 (대기중 → 준비중 → 완료)
- 주문 삭제 및 총액 재계산
- SSE 이벤트 발행 (주문 생성/상태 변경 시)

### 9. TableService
**목적**: 테이블 세션 비즈니스 로직 처리
**책임**:
- 테이블 세션 생성/종료 로직
- 이용 완료 처리 (현재 주문 → 이력 이동)
- 세션 만료 처리
- 과거 주문 이력 조회

### 10. SseService
**목적**: SSE 이벤트 브로드캐스트 관리
**책임**:
- SSE Emitter 풀 관리 (관리자/고객 분리)
- 이벤트 발행 (신규 주문, 상태 변경)
- 연결 만료/오류 처리

### 11. JwtFilter (Security Filter)
**목적**: HTTP 요청 인증 필터
**책임**:
- Authorization 헤더에서 JWT 추출 및 검증
- 관리자 요청 인증
- 테이블 세션 토큰 검증
- SecurityContext 설정

### 12. Repository Layer
**목적**: 데이터 접근 추상화
**컴포넌트**:
- `StoreRepository`: 매장 정보 CRUD
- `TableRepository`: 테이블 정보 CRUD
- `TableSessionRepository`: 세션 관리
- `CategoryRepository`: 카테고리 CRUD
- `MenuRepository`: 메뉴 CRUD (순서 포함)
- `OrderRepository`: 주문 CRUD
- `OrderItemRepository`: 주문 항목 CRUD

## Frontend - Customer App Components (React)

### 1. App (Root)
**목적**: 앱 진입점, 라우팅, 전역 상태 관리
**책임**:
- React Router 설정
- 자동 로그인 초기화 (localStorage 확인)
- 전역 상태 (테이블 세션, 장바구니) 관리

### 2. AutoLoginGuard
**목적**: 자동 로그인 처리 및 라우팅 보호
**책임**:
- localStorage에서 인증 정보 로드
- 서버 자동 로그인 API 호출
- 인증 실패 시 초기 설정 화면으로 리다이렉트

### 3. SetupPage
**목적**: 테이블 초기 설정 화면 (관리자 1회 수행)
**책임**:
- 매장 식별자, 테이블 번호, 비밀번호 입력 폼
- 설정 정보 localStorage 저장

### 4. MenuPage
**목적**: 메뉴 탐색 기본 화면
**책임**:
- 카테고리 탭 렌더링
- 메뉴 카드 그리드 렌더링
- 장바구니 담기 액션 처리

### 5. MenuCard
**목적**: 개별 메뉴 카드 컴포넌트
**책임**:
- 메뉴 이미지, 이름, 가격 표시
- 수량 선택 및 장바구니 추가 버튼

### 6. CartDrawer / CartPage
**목적**: 장바구니 UI
**책임**:
- 담긴 메뉴 목록 및 수량 표시
- 수량 증감, 항목 삭제
- 총 금액 실시간 계산
- 주문하기 버튼

### 7. OrderConfirmPage
**목적**: 주문 확정 화면
**책임**:
- 주문 내역 최종 확인
- 주문 API 호출
- 성공 시 5초 카운트다운 후 메뉴 화면 리다이렉트

### 8. OrderHistoryPage
**목적**: 현재 세션 주문 내역 화면
**책임**:
- 현재 세션 주문 목록 표시
- SSE 연결로 주문 상태 실시간 업데이트

### 9. useCart (Custom Hook)
**목적**: 장바구니 상태 관리
**책임**:
- localStorage 기반 장바구니 상태 관리
- 메뉴 추가/삭제/수량 조절/비우기

### 10. useOrderSse (Custom Hook)
**목적**: 고객용 SSE 연결 관리
**책임**:
- SSE 연결 수립 및 유지
- 주문 상태 변경 이벤트 수신
- 자동 재연결 처리

## Frontend - Admin App Components (React)

### 1. App (Root)
**목적**: 관리자 앱 진입점, 라우팅, 인증 상태 관리
**책임**:
- React Router 설정
- JWT 토큰 관리 (localStorage)
- 인증 상태 전역 관리

### 2. LoginPage
**목적**: 관리자 로그인 화면
**책임**:
- 매장 식별자, 사용자명, 비밀번호 입력 폼
- 로그인 API 호출 및 JWT 저장
- 로그인 실패 에러 표시

### 3. AuthGuard
**목적**: 인증된 사용자만 접근 허용
**책임**:
- JWT 토큰 유효성 확인
- 미인증 시 로그인 페이지 리다이렉트
- 토큰 만료 감지 및 자동 로그아웃

### 4. DashboardPage
**목적**: 실시간 주문 모니터링 메인 화면
**책임**:
- 테이블별 카드 그리드 렌더링
- SSE 연결로 실시간 주문 업데이트
- 신규 주문 시각적 강조

### 5. TableCard
**목적**: 테이블별 주문 현황 카드
**책임**:
- 테이블 번호, 총 주문액, 최신 주문 미리보기 표시
- 클릭 시 상세 모달 오픈
- 신규 주문 애니메이션

### 6. TableDetailModal
**목적**: 테이블 주문 상세 모달
**책임**:
- 전체 주문 목록 표시
- 주문 상태 변경 버튼
- 주문 삭제 버튼
- 이용 완료 처리 버튼

### 7. TableManagePage
**목적**: 테이블 초기 설정 및 관리 화면
**책임**:
- 테이블 초기 설정 폼
- 과거 주문 내역 조회 (날짜 필터)

### 8. MenuManagePage
**목적**: 메뉴 관리 화면
**책임**:
- 카테고리별 메뉴 목록 표시
- 메뉴 추가/수정/삭제 폼
- 이미지 파일 업로드
- 메뉴 순서 조정

### 9. useAdminSse (Custom Hook)
**목적**: 관리자용 SSE 연결 관리
**책임**:
- SSE 연결 수립 및 유지
- 신규 주문/상태 변경 이벤트 수신
- 자동 재연결 처리

### 10. useAuth (Custom Hook)
**목적**: 인증 상태 및 JWT 관리
**책임**:
- JWT 토큰 저장/조회/삭제
- 토큰 만료 시간 확인
- 로그인/로그아웃 액션
